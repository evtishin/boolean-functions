#цнт
         Список основных определений и понятий.
     \блф Булева функция\
     \едн Единичные\ и \ссд Соседние\ наборы
     \фкт Фиктивные\ и \сщн Существенные\ переменные
     \спр Суперпозиция функций\
     \днф Дизъюнктивная нормальная форма\
     \сдн Совершенная ДНФ\
     \сок Сокращенная ДНФ\
     \имп Импликанта\
     \элк Элементарная конъюнкция\
     \тпк Тупиковая ДНФ\
     \мнм Минимизация ДНФ\
     \кнк Конъюктивная нормальная форма\
     \жгл Полином Жегалкина\
#блф
    Булевой функцией называется функция
  f(x1,x2,...,xN) ,принимающая значения 0 ,1
  и аргументы которой принимают значения 0 ,1.
  \цнт  \
#едн
    Наборы значений аргументов ,на которых
  функция равна 1 называются единичными
  наборами функции ,а наборы ,на которых функция
  равна 0 ,будем называются нулевыми наборами
  данной булевой функции.
    См. также      \блф Булева функция\
  \цнт  \
#ссд
    Наборы u и v значений переменных называются
  соседними по i-той переменной,если отличаются
  только i-той координатой ,то есть имеют вид:
    u = (a1,a2,...,a(i-1),0,a(i+1),...,aN)
    v = (a1,a2,...,a(j-1),1,a(j+1),...,aN).
    См. также      \блф Булева функция\
  \цнт  \
#фкт
    Переменная x(i) называется фиктивной
  переменной булевой функции
  f(x1,...,x(i),...,xN) ,если для любых наборов
  u ,v значений переменных ,соседних по i-той
  переменной ,выполнится равенство: f(u)=f(v).
    См. также      \блф Булева функция\
                   \сщн Существенные переменные\
  \цнт  \
#сщн
    Переменная x(i) называется существенной
  переменной булевой функции
  f(x1,...,x(i),...,xN) ,если существует хотя бы
  одна пара u ,v наборов значений переменных ,
  соседних по i-той переменной ,такая ,
  что  f( u )  не равно  f( v ).
    См. также      \блф Булева функция\
                   \фкт Фиктивные переменные\
  \цнт  \
#спр
    Суперпозицией функций f1,...,fN называется
  функция f ,полученная с помощью подстановок
  этих функций друг в друга и переименованием
  переменных ,а формулой называется выражение,
  описывающее эту суперпозицию.
    См. также      \блф Булева функция\
  \цнт  \
#элк
    Элементарной коньюнкцией называется
  коньюнкция переменных или их отрицаний ,
  в которой каждая переменная встречается
  не более одного раза.
  \цнт  \
#днф
    Дизъюнктивной нормальной формой ( ДНФ )
  называется формула ,имеющая вид дизъюнкции
  элементарных коньюнкций. Каждой ДНФ
  соответствует некотороя булева функция.
    См. также      \блф Булева функция\
                   \элк Элементарная конъюнкция\
  \цнт  \
#сдн
    Совершенной дизъюнктивной нормальной формой
  ( СДНФ ) данной булевой функции f(x1,...,xN)
  называется такая ее ДНФ , у которой:
  - все элементарные коньюнкции различны;
  - каждая элементарная конъюнкция имеет N букв,
  где каждая буква - либо переменная из
  множества { x1,...,xN } ,либо переменная
  с отрицанием.
    См. также      \блф Булева функция\
                   \днф ДНФ\
                   \элк Элементарная конъюнкция\
  \цнт  \
#мнм
    ДНФ данной функции называется минимальной
  ДНФ ,если она содержит наименьшее возможное
  число букв среди всех ДНФ данной функции f.
    См. также      \блф Булева функция\
                   \днф ДНФ\
  \цнт  \
#имп
    Элементарная конъюнкция E называется
  импликантой булевой функции f,если E => f = 1.
  Из определения следует ,что элементарная
  конъюнкция E является импликантой тогда и
  только тогда ,когда ME включено в Mf.
    См. также      \блф Булева функция\
                   \элк Элементарная конъюнкция\
  \цнт  \
#при
    Импликанта E булевой функции f называется
  простой импликантой булевой функции f ,если
  после удаления любой буквы из E она перестает
  быть импликантой функции f.
    См. также      \блф Булева функция\
                   \имп Импликанта\
  \цнт  \
#сок
    ДНФ данной функции f называется сокращенной
  ДНФ ,если она представляет из себя дизъюнкцию
  всех простых импликант данной булевой
  функции f.
    См. также      \блф Булева функция\
                   \имп Простая импликанта\
                   \днф ДНФ\
  \цнт  \
#тпк
    ДНФ данной функции f называется тупиковой ,
  если она состоит из простых импликант функции,
  причем удаление любой конъюнкции приводит к
  ДНФ , не равносильной данной функции.
    См. также      \блф Булева функция\
                   \имп Простая импликанта\
                   \днф ДНФ\
  \цнт  \
#кнк
    Любая булева функция,не равная тождественно
  1,представима своей совершенной конъюктивной
  нормальной формой (СКНФ)
  по формуле:        __a1        __aN
  f(x1,...,xN) = KN (x1  DS...DS xN  ) ,
  где конъюнкции берутся по нулевым наборам
  данной функции.
    См. также      \блф Булева функция\
  \цнт  \
#жгл
    Полиномом Жегалкина булевой функции f
  называется ее представление в виде суммы по
  модулю 2 элементарных конъюнкций.
    См. также      \блф Булева функция\
                   \элк Элементарная конъюнкция\
  \цнт  \
#end
